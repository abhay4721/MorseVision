1. **Create Virtual Environment**
   ```bash
   python -m venv venv
   ```

   Activate it:

   - **Linux / macOS**
     ```bash
     source venv/bin/activate
     ```

   - **Windows**
     ```bash
     venv\Scripts\activate
     ```

2. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```
